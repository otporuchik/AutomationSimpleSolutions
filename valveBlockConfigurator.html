  • Описание изделия

    • Традиционный клапанный блок (код = 0104)
    • Интегральный клапанный блок (код = 0105)
    • Штуцерный клапанный блок (код = 0106)

  • Изготовитель

    Метран

    Без вариантов.

  • Тип исполнения клапанного блока

    • Традиционный, соединение типа фланец-фланец (код = T)
    • Компактный* (код = W)
    • Coplanar (код = C)
    • Coplanar с боковым подключением (код = S)
    • Традиционный (код = T)
    • Резьбовое (соединение типа резьба-резьба) (код = T)

    * - не применяется для двухвентильных блоков (код 2) и для кода присоединения к процессу = F (фланцевое).

  • Количество вентилей

    • Одновентильный (код = 1)
    • Двухвентильный (код = 2)
    • Трехвентильный (код = 3)
    • Пятивентильный (код = 5)

  • Исполнение по материалам

    • Нержавеющая сталь 316 SST или 12Х18Н10Т (код = 2)
    • Нержавеющая сталь 316 SST или сплав С-276 (код = 4)

  • Соединение с процессом

    • Резьбовое соединение 1/2 NPT наружная (код = A)
    • Резьбовое соединение 1/2 NPT внутренняя (код = B)
    • Резьбовое соединение М20х1,5 наружная (код = C)
    • Резьбовое соединение К 1/2" ГОСТ 6111 внутренняя* (код = D)
    • Фланцевое соединение (код = F)
    • Фланцевое соединение и резьбовое соединение 1/4 NPT внутренняя (код = F)
    • Резьбовое соединение 1/4" ГОСТ 6111 внутренняя (код = E)

    * - Допускается замена на код B соединения с процессом.

  • Тип соединения с датчиком

    • Резьбовое соединение с наружной резьбой 1/2 NPT (код = A)
    • Накидная гайка М20х1,5 (для прямого подключения) (код = B)
    • Резьбовое соединение с внутренней резьбой М20х1,5 (код = C)

  • Материал уплотнения вентиля*

    Материал уплотрения вентиля

    • Фторопласт (PTFE) (код = 1)
    • Графит (код = 2)
    • Графитонаполненный фторопласт (Ghraphite-filled PTFE) (только для кода специальной очистки UC - очистка для применения в среде, содержащей газообразный кислород.) (код = 3)

    * - Материал уплотнительных колец соединения клапанного блока с датчиком давления соответствует коду материала уплотнения вентиля клапанного блока.

  • Тип датчика для подсоединения

    • С традиционным фланцем (код = 1)
    • С традиционным фланцем для высокого давления (код типа исполнения клапанного блока = W = компактный) (код = 2)

  • Седло клапанного блока

    • Интегральное
    • Обычное

  • Специальная очистка

    • Поставляется с установленным клапанным блоком* (не поставляется для датчиков с кодом ОР**). (код = S5)
    • Нет

  • Специальное исполнение

    • Исполнение с дренажными клапанами (код = H1)
    • Исполнеие для подключения метрологического оборудования = штуцер М10х1,5, установленный в дренажные отверстия, максимальное давление штуцера 16МПа. (код H2)

  • Дополнительные опции

    • Специальное исполнение

      • Гарантийный срок эксплуатации клапанных блоков - 5 лет (код = WR5)
      • Гарантийный срок эксплуатации клапанных блоков - обычный (код = -)

    • Монтажные кронштейны

      • Кронштейн из углеродистой стали для крепления клапанного блока на трубе (код = VC)
      • Кронштейн из нержавеющей стали для крепления клапанного блока на трубе (код = VS)
      • Монтажный кронштейн для крепления клапанного блока с кодом типа исполнения клапанного блока = Т (традиционного типа) на трубе с наружным диаметром 60мм (материал кронштейна и болтов - углеродистая сталь с покрытием) (код = B1)
      • Монтажный кронштейн для крепления клапанного блока с кодом типа исполнения клапанного блока = С (Coplanar) и S (Coplanar с боковым подключением) на панели или трубе с наружным диаметром 60мм (материал кронштейна и болтов - нержавеющая сталь) (код = B4)

    • Монтажные части*

      • Ниппель с накидной гайкой М20х1,5, для соединения по наружному диаметру трубы 14мм (код = D0**)
      • Монтажный фланец из нержавеющей стали 316 с резьбовым отверстием К1/4" ГОСТ 6111 (код = D1)
      • Монтажный фланец из нержавеющей стали 316 с резьбовым отверстием К1/2" ГОСТ 6111 (код = D2)
      • Монтажный фланец из нержавеющей стали 316 с резьбовым отверстием 1/4 NPT (код = D3)
      • Монтажный фланец из нержавеющей стали 316 с резьбовым отверстием 1/2 NPT (код = D4)
      • Монтажный фланец из нержавеющей стали 316 с ниппелем, с накидной гайкой М20х1,5, для соединения по наружному диаметру трубы 14мм (код = D5**)
      • Монтажный фланец из углеродистой стали с покрытием, с ниппелем для соединения по наружному диаметру трубы 14мм (код = D6)

      * - Материал болтов крепления монтажных фланцев к клапанному блоку - углеродистая сталь с покрытием. Материал уплотнительных колец монтажных фланцев соответствует коду материала уплотнения вентиля клапанного блока.

      ** - Материал накидной гайки - углеродистая сталь с покрытием, материал уплотнительной прокладки - медь для кодов материала ниппеля 3 и 5, нержавеющая сталь 12Х18Н10Т для кода материала ниппеля 2.
      Код материала ниппеля:
      2 - Нержавеющая сталь 12Х18Н10Т или 316L;
      3 - Углеродистая сталь с покрытием;
      5 - Углеродистая сталь низкотемпературная 09Г2С с покрытием.

      Монтажные части

      • Монтажный фланец из нержавеющей стали 316 с ниппелем, с накидной гайкой М20х1,5, для соединения по наружному диаметру трубы 14мм (код = D5*)
      • Ниппель с накидной гайкой М20х1,5, для соединения по наружному диаметру трубы 14мм (код = 2F**)

      * - Применяется для датчиков типа Метран-150 моделей 150CG или 150CGR с традиционным фланцем.

      ** - Применяется для датчиков типа Метран-150 моделей 150TA, 150TG, 150TAR или 150TGR.

    • Материал ниппеля

      • Нержавеющая сталь 12Х18Н10Т или 316L (код = 2)
      • Углеродистая сталь с покрытием (код = 3)
      • Углеродистая сталь низкотемпературная 09Г2С с покрытием (код = 5)

    • Болты для сборки с датчиком давления

      • Болт с резьбой М10х1,5 из углеродистой стали с покрытием (код = L3)
      • Болт с резьбой 7/16-20UNF из нержавеющей стали ASTMA A193 B8M CLASS2 (только для кода типа исполнения клапанного блока W* только для кода типа датчика для подсоединения 2**) (код = L8)

      * - Код типа исполнения клапанного блока W = компактный, соединение типа фланец-фланец.

      ** - Код типа датчика для подсоединения 2 = с традиционным фланцем для высокого давления.

    • Болты для сборки с датчиком давления

      • Болты 7/16-20UNF из углеродистой стали по ASTMA A 449 Type1 с покрытием (код = L3)
      • Болты 7/16-20UNF из нержавеющей стали 316 SST по ASTM F593 GP2 CW (код = L4)
      • Болты 7/16-20UNF из нержавеющей стали ASTM A193 B8M Class2 (для кода специального исполнения BR6*, для сборки с датчиком давления с кодом HP**) (код = L8)

      * - BR6 = Температура окружающей среды от минус 600С.

      ** - НР = Предельно допускаемое рабочее давление 35МПа (не применяется с датчиками CD, CG, TG, TA).
