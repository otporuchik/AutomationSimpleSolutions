  • Модель датчика:

    • 150TG - датчик избыточного давления
    • 150TA - датчик абсолютного давления

  • Диапазон измерений

    Код Датчик 150TG Датчик 150TA
    1 (-101.3)-160кПа 0-160кПа
    2 (-101.3)-1.0МПа 0-1.0МПа
    3 (-101.3)-6.0МПа 0-6.0МПа
    4 (-101.3)-25МПа 0-25МПа
    5 (-101.3)-60МПа -

    101.3кПа - предполагаемое атмосферное давление, принимается как нулевая отметка.

  • Технологическое соединение

    • М20х1.5 (код = 2G)
    • 1/2NPT, внутренняя резьба (код = 2B)

  • Материал разделительной мембраны

    Материал деталей, контактирующих с рабочей средой

    • 315L SST (код = 2)
    • Hastelloy (код = 3)

  • Заполняющая жидкость

    (длина погружной части, зависит от выбранного кода исполнения защитной арматуры.)

    • Силикон (код = 1)
    • Нет

  • Выходной сигнал

    • 4-20мА с цифровым сигналом на базе протокола HART (код = A)
    • 0-5мА (поставляется только с кодом М5 (встроенный ЖКИ), не применяется для датчиков с кодом IM (сертификация искробезопасности 0Ex ia IIC T5 GaX) (код = В)

  • Индикация

    • Встроенный ЖКИ* (код = М5)
    • Нет

    * - Жидкокристаллический индикатор.

  • Для специального применения

    • Сертификация искробезопасности 0Ex ia IIC T5 Ga X (код = IM)
    • Сертификация взрывобезопасности 1Ex db IIC T6 Gb X (код = EM)
    • Искро/взрывозащиты нет

  • Встроенные клапанные блоки

    • Поставляется с установленным клапанным блоком* (не поставляется для датчиков с кодом ОР**). (код = S5)
    • Нет

    * - Клапанный блок оформляется отдельно строкой, обозначение в соответствии с ТУ 3742-057-51453097-2009. При заказе датчика с кодом S5 монтажные части не указываются в обозначении датчика. Монтажные части указываются в обозначении клапанного блока.

    ** - код ОР - для применения на предприятиях "Газпром добыча Оренбург" (поставляется только с кодом разделительной мембраны и деталей, контактирующих с рабочей средой - 2 = 316L SST, без монтажных частей, без клапанного блока.

  • Монтажные части (только для кода технологического соединения 2G, не поставляются для датчиков с кодом ОР*.

    • Переходники с резьбой 1/4 NPT внутренней (код = 2A)
    • Переходники с резьбой 1/4 NPT наружной (код = 2D)
    • Переходники с резьбой 1/2 NPT наружной (код = 2E)
    • Ниппель (внутренний ⌀8) с накидной гайкой М20х1,5 (код = 2F)
    • Ниппель (внутренний ⌀10) с накидной гайкой М20х1,5 (код = 2H)

    * - код ОР - для применения на предприятиях "Газпром добыча Оренбург" (поставляется только с кодом разделительной мембраны и деталей, контактирующих с рабочей средой - 2 = 316L SST, без монтажных частей, без клапанного блока.

  • Материал ниппеля* (только для кодов монтажных частей 2F**, 2H***)

    • Сталь 12Х18Н10Т или 316L SST (код = 2)
    • Углеродистая сталь с покрытием (код = 4)
    • Углеродистая сталь 09Г2С с покрытием (код = 5)

    * - Материал уплотнительной прокладки - сталь 12Х18Н10Т для кода материала ниппеля 2 , медь - для кодов материала ниппеля 4 (углеродистая сталь с покрытием) и 5 (углеродистая сталь 09Г2С). Материал накидной гайки для кода 2F, 2H - углеродистая сталь с покрытием.

    ** - 2F - Ниппель (внутренний ⌀8) с накидной гайкой М20х1,5

    *** - 2H - Ниппель (внутренний ⌀10) с накидной гайкой М20х1,5

  • Монтажные кронштейны

    • Монтажный кронштейн для крепления на панели (материал - углеродистая сталь с покрытием) (код = В1)
    • Монтажный кронштейн для крепления на панели (материал - сталь 12Х18Н10Т) (код = В4)

  • Гарантийный срок эксплуатации

    • Гарантийный срок эксплуатации датчиков - 5 лет (код = WR5)
    • Гарантия не имеет значения

    • Маркировочная табличка по заказу потребителя (необходимо заполнить лист параметров настройки, код С1 в заказе не указывают) (код = ST)

    • Кабельный ввод (нажмите, чтобы перейти в таблицу подбора)

    Штепсельные разъемы датчиков метран 150

    Кабельные вводы датчиков метран 150 согласно каталогу датчиков давления метран (выпуск 2020)

    Кабельные вводы предназначены для фиксации различных типов кабелей при подключении датчиков с целью защиты от попадания внутрь корпуса влаки и пыли. Кабельные вводы (кроме К01, К04, К05, К09) имеют взрывозащищенныое исполнение с маркировкой взрывозащиты ExdIIC.

    Кабельные вводы поставляются в комплекте с датчиком.

    Диаметр кабеля может быть изменен в незначительных пределах.

    Степень защиты от воздействия пыли и воды IP66 по ГОСТ 14254.

    Диапазон температур окружающей среды для металлических кабельных вводов от -60 до 130°С, для кабельных вводов из полиамида от -20 до 70°С.
